_scorer(self, clf, X, y)

source code 

Function that scores a classifier according to what is available as a predict function. Input:

  • clf = Fitted classifier object
  • X = input data matrix
  • y = estimated labels

Output:

  • AUC score for binary classification or F1 for multiclass

The order of priority is as follows:

  • predict_proba
  • decision_function
  • predict

_reg_scorer(self, reg, X, y, scoring)

source code 

Function that scores a regressor according to what is available as a predict function. Input:

  • reg = Fitted regressor object
  • X = input data matrix
  • y = corresponding values to the data matrix

Output:

  • The mean sqaure error or r squared value for the given regressor and data. The default scoring is r squared value.
